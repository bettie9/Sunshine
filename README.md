# Sunshine ☀️

Sleek League of Legends companion — skin changer (with chromas + multi-stage forms), auto-accept, auto-pick, and lobby revealer.

## Install

```powershell
irm https://raw.githubusercontent.com/bettie9/Sunshine/main/install.ps1 | iex
```

Pulls the latest signed installer from this repo's releases and runs it silently. UAC prompt will appear.

## Uninstall

```powershell
irm https://raw.githubusercontent.com/bettie9/Sunshine/main/uninstall.ps1 | iex
```

Pass `-PurgeData` (or answer **y** at the prompt) to also wipe local skins, hashes, and configuration.
